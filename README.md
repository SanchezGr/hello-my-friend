<h1> Вебзастосування "Привіт, мій друже!" </h1>
<p> Вебзастосування дає змогу вивести привітальне повідомлення "Привіт, мій друже!" по центру екрану. Вебзастосування створено за допомогою НТМL і CSS.</p>
<p> Переглянути вебзастосування можна за гіперпосиланням: <a href="https://sanchezgr.github.io/hello-my-friend/"> https://sanchezgr.github.io/hello-my-friend/ </a> </p>
<p> Для перегляду вебзастосування локально на комп’ютері послідовно виконайте наступні команди:</p>
<ol>
  <li> <code> git clone https://github.com/SanchezGr/hello-my-friend.git </code>, щоб клонувати репозиторій на локальний комп’ютер; </li>
  <li> <code> cd hello-my-friend </code>, щоб перейти в папку hello-my-friend; </li>
  <li> <code> start index.html </code>, щоб переглянути вебзастосування у браузері. </li>
</ol>
